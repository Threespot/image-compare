Image Compare

image-compare is a web component to help compare two images with a visibility slider.

After including the script (TODO) it can be used like so: 


  <image-compare label-text="Use the slider to control the visibility of the two images.">
    <img slot="image-1" alt="The Longhorn Steakhouse home page with a headline saying 'Steak as it was meant to be'" src="https://29comwzoq712ml5vj5gf479x-wpengine.netdna-ssl.com/wp-content/uploads/2021/06/longhorn-screenshot-750.jpg"/>
    <img slot="image-2" alt="The Olive Garden home page with a headline saying 'The comfort you crave'" src="https://29comwzoq712ml5vj5gf479x-wpengine.netdna-ssl.com/wp-content/uploads/2021/06/olivegarden-screenshot-750.jpg"/>
  </image-compare>

It expects two images to be passed into the image-1 and image-2 slots.

There is an optional label-text attribute which can be used to provide extra context to screen reader users and other assistive technologies. If left unset it will default to the following:

Control how much of each overlapping image is shown. 0 means the first image is completely hidden and the second image is fully visible. 100 means the first image is fully visible and the second image is completely hidden. 50 means both images are half-shown, half-hidden.
